Clique com o mouse e digite os números de 0 a 9 para mudar de cor. Para desenhar uma reta, aperte 'r'. Para desenhar um triângulo, aperte 't'. Para alterar a espessura, aperte 'e'. Para alterar a cor, aperte 'k'. Lembre-se que sempre que você apertar 'r' ou 't' o modo default é o de mudar cor, assim sempre que quiser mudar a espessura de uma nova figura, aperte 'e'.
